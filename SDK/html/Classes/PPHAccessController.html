Overview

PPHAccessController helps you build OAuth authentication into your app. PLEASE NOTE: In order to implement proper security, you should create a server similar to the sample server included in this SDK to store your app secret server-side (only).

If you have authentication other than PayPal Access to your server, use that to further lock down who your server will give out access tokens to.
