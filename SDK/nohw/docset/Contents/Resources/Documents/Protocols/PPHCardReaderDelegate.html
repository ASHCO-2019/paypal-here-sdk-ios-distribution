PPHCardReaderDelegate Protocol Reference

Conforms to NSObject
Declared in PPHCardReaderDelegate.h

Overview

Basic events for audio, feature port, and EMV readers. While methods are marked optional for cases where you just want to see certain events (like reader software upgrade), the delegate that’s actually looking to complete a payment is going to need all of these.
