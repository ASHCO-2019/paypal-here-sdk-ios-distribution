isBackgroundBeaconEnabled

Using background beacon services requires specific entitlements for your application. Since we cannot read these entitlements automatically at runtime, you must tell the SDK if you want background beacon advertisements to be enabled. Defaults to false. Must be set before setBeaconLocation as it generally only matters when starting CoreBluetooth services.

@property (nonatomic, assign) BOOL isBackgroundBeaconEnabled
