Inherits from NSObject
Conforms to NSCoding
Declared in PPHAccessAccount.h

Overview

The PPHAccessUser class stores information about a successfully authenticated PayPal Access account. Via the NSCoding protocol, this object can be saved to persistent storage.

PLEASE NOTE: The tokens in this object are sensitive information giving the bearer the ability to perform operations against the PayPal account, and must be treated accordingly (thus stored securely). For simplicity, the PayPalHereSDK singleton has methods to do this for you.
