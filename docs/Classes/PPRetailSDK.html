PPRetailSDK Class Reference

Inherits from PPRetailObject : NSObject
Declared in PPRetailSDK.h

Overview

The PayPal Here SDK object is the main entry point for all SDK operations. Because we provide native-specific versions of the highest level interface, this class is essentially used as a helper for binding top level events (such as a new card reader) to the native partner objects.
